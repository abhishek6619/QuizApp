# Java Quiz Application

A simple Java Swing-based quiz app for beginners.

## Features

- 5 multiple-choice questions
- Tracks score
- Shows result at the end
- User-friendly interface

## How to Run

1. Compile all `.java` files inside `src/`
2. Run `QuizMain.java`

## Tools

- Java 8+
- Swing for UI
